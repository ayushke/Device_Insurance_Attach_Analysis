# Device Insurance Attach Rate Analysis – Jumbo & Company

## Project Overview
This project analyzes device insurance attach rates for Jumbo & Company across multiple branches and stores. The objective is to understand month-wise and store-wise trends, identify high and low performing stores, and provide actionable insights to improve attach rate performance.

Additionally, based on historical data, attach rates for January have been predicted at the store level.

## Dataset Description
The dataset contains monthly attach percentages of device insurance plans (such as screen protection, extended warranty, and accidental damage) against the number of devices sold across different retail stores.

Time period covered:
- August to December

Key fields:
- Branch
- Store Name
- Monthly Attach Percentage

## Analysis Performed
- Month-wise attach rate trend analysis
- Branch-wise performance comparison
- Store-level performance evaluation
- Categorization of stores into High, Medium, and Low performance groups
- Identification of top-performing and bottom-performing stores
- January attach rate prediction at store level using historical averages
